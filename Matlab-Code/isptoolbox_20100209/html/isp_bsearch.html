ISP_BSEARCH

Binary search

Syntax

idx=isp_bsearch(borders, val)

Description

Find the index 'idx' in 'borders' such that borders(idx) <= val < borders(idx+1)

Input

borders:
Sorted array of border values.
val:
The value to search for. If 'val' is an array, 'idx' will also be an array of the same size.

Output

idx:
Index of the element or interval in 'borders' that match 'val'. If val < borders(1), idx=0 is returned, and if borders(end) <= val, idx=length(borders)+1 is returned.

Example

borders=round(sort(rand(1,10)*100)) isp_bsearch(borders, 0.6)
