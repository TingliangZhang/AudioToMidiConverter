ISP_NONEMPTY

Return the first non-empty input argument

Syntax

arg=isp_nonempty(input1, input2, ...)

Description

Returns the first non-empty input argument. It is intended for use in situations where an empty value should be replaced by a default value. It is particularly useful with isp_interpretarguments when some default values depend on other input arguments.

Input

input1, input2, ...:
Any inputs.

Output

arg:
The first non-empty input argument. If all input arguments are empty, the last argument is returned.

Example

function myplot(x, y, varargin)
% Allow user to specify empty x:
x = isp_nonempty(x, 1:length(y));
plot(x,y)

isp_interpretarguments.
