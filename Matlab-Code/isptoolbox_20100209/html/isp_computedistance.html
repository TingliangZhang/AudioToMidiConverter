Compute distance matrix.

Syntax

[distancematrix, comptime] = isp_computedistance(distance, features1, features2)

Description

Computes the distance matrix from features previously extracted.

Input

distance:
Struct defining a distance measure as returned by e.g. isp_mfccgmmkl. This function uses the 'options' and either the 'computedistancematrix' or 'computedistance' fields.
features1, features2:
Cell array of features.

Output

distancematrix:
Matrix where entry (m, n) is the distance between features1{m} and features2{n}
comptime:
Computation time.

See also

isp_evaluate, isp_extractfeature.
